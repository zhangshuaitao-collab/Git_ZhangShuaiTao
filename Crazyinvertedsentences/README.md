# 文字排序游戏 - 倒装句挑战

一个有趣的H5文字排序游戏，帮助玩家学习和理解中文倒装句的语法结构。

## 🎮 游戏特点

- **十个精心设计的关卡**：从简单到复杂，循序渐进
- **直观的拖拽操作**：通过拖动文字或词组重新排列句子
- **实时计时系统**：挑战您的速度和效率
- **本地排行榜**：记录您的最佳成绩
- **精美的UI设计**：现代化界面，流畅的动画效果
- **响应式设计**：支持PC和移动设备

## 🎯 游戏玩法

1. **目标**：将正常语序的句子改造成倒装句
2. **操作**：拖动下方的文字块到上方的排序区域
3. **过关**：正确排列后点击"检查答案"
4. **计时**：从第一关开始计时，全部通关后显示总用时
5. **挑战**：尝试获得更好的成绩并进入排行榜

## 📚 关卡内容

游戏包含10个关卡，涵盖不同类型的倒装句：

1. **时间状语前置**：强调时间概念
2. **地点状语前置**：强调地点概念
3. **原因状语前置**：强调原因
4. **目的状语前置**：强调目的
5. **条件状语前置**：强调条件
6. **让步状语前置**：强调转折
7. **让步复句变换**：复杂让步关系
8. **充分条件表达**：条件关系变换
9. **方式状语前置**：强调方式方法
10. **综合状语运用**：复合状语结构

## 🚀 运行游戏

### 方法一：直接打开
1. 下载所有文件到本地
2. 用浏览器直接打开 `index.html`

### 方法二：本地服务器
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 🎵 音效支持

游戏框架已预留音效接口，支持以下音效：
- 拖拽放置音效
- 答案正确音效
- 答案错误音效
- 游戏完成音效
- 计时音效

要启用音效，请在 `sounds/` 目录下放置相应的音频文件：
- `drop.mp3` - 拖拽音效
- `success.mp3` - 成功音效
- `error.mp3` - 错误音效
- `complete.mp3` - 完成音效
- `tick.mp3` - 计时音效
