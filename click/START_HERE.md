# 🚀 快速开始 - 快速点击助手桌面版

## 🎯 三分钟快速上手指南

欢迎使用**快速点击助手桌面版**！这是一个强大的桌面自动点击软件，可以点击任何软件窗口中的按钮。

---

## ⚡ 立即开始使用

### 方式一：一键启动（推荐）

#### Windows用户：
```bash
# 双击运行
快速点击助手.bat
```

#### Linux/macOS用户：
```bash
# 终端运行
./快速点击助手.sh
```

### 方式二：Python直接运行

```bash
# 1. 自动安装依赖
python install.py

# 2. 启动程序  
python main.py
```

## 📋 快速操作流程

### 🎯 第一步：选择目标窗口
1. 点击程序中的 **"选择窗口"** 按钮
2. 从弹出的窗口列表中选择目标软件
3. 确认选择

### 📍 第二步：选择点击坐标
1. 点击 **"选择坐标"** 按钮  
2. 在目标软件的截图上点击要自动点击的位置
3. 确认坐标

### ⚙️ 第三步：设置参数
- **点击间隔**: 1000毫秒（推荐）
- **点击次数**: 0 = 无限点击
- **鼠标键**: 左键（默认）

### ▶️ 第四步：开始点击
1. 点击 **"开始点击"** 按钮
2. 或按快捷键 `Ctrl+Shift+Space`

### ⏹️ 第五步：停止点击
1. 点击 **"停止点击"** 按钮
2. 或按快捷键 `Ctrl+Shift+Space`
3. 或按 `ESC` 键紧急停止

## 🆘 常见问题解决

### ❌ 程序无法启动？
1. **检查Python**: 确保安装Python 3.6+
2. **安装依赖**: 运行 `python install.py`
3. **重新启动**: 重启电脑后再试

### ❌ 无法选择窗口？
1. **目标可见**: 确保目标软件窗口可见
2. **管理员权限**: 尝试以管理员身份运行
3. **防病毒软件**: 检查是否被拦截

### ❌ 点击不生效？
1. **坐标准确**: 重新选择点击坐标
2. **窗口活跃**: 确保目标窗口未最小化
3. **点击类型**: 尝试不同的鼠标键

### ❌ 程序运行缓慢？
1. **调整间隔**: 适当增加点击间隔
2. **关闭程序**: 关闭不必要的后台程序
3. **重启程序**: 重新启动点击助手
