# 🖱️ 快速点击助手 (Auto Click Helper)

一个强大的浏览器扩展，用于自动点击网页按钮，提高重复操作的效率。

## ✨ 主要功能

- **智能元素选择** - 可视化选择要点击的按钮或元素
- **批量点击支持** - 同时点击多个元素
- **灵活的点击设置** - 自定义点击间隔、次数和类型
- **多种点击模式** - 普通点击、双击、按下、释放
- **配置保存/加载** - 保存常用的点击配置
- **键盘快捷键** - `Ctrl+Shift+Space` 快速开始/停止
- **实时状态显示** - 点击计数和状态监控
- **美观的现代界面** - 直观易用的用户界面

## 🚀 安装步骤

### 1. 下载扩展文件
将所有文件下载到本地文件夹。

### 2. 在浏览器中加载扩展

#### Chrome/Edge 浏览器:
1. 打开浏览器，访问 `chrome://extensions/` (Chrome) 或 `edge://extensions/` (Edge)
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择包含扩展文件的文件夹
5. 确认安装

#### Firefox 浏览器:
1. 访问 `about:debugging`
2. 点击"此Firefox"
3. 点击"临时载入附加组件"
4. 选择 `manifest.json` 文件

### 3. 添加扩展图标 (可选)
你可以在扩展文件夹中添加以下图标文件：
- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)  
- `icon128.png` (128x128 像素)

## 📖 使用指南

### 基本使用流程

1. **打开扩展**
   - 点击浏览器工具栏中的扩展图标
   - 或使用快捷键 `Ctrl+Shift+Space`

2. **选择要点击的元素**
   - 点击"选择要点击的按钮"
   - 在网页中点击你想要自动点击的元素
   - 选择的元素会被高亮显示

3. **配置点击设置**
   - **点击间隔**: 设置两次点击之间的时间间隔（毫秒）
   - **点击次数**: 设置总点击次数（0表示无限点击）
   - **点击类型**: 选择点击类型（普通点击、双击等）

4. **开始自动点击**
   - 点击"开始点击"按钮
   - 或使用快捷键 `Ctrl+Shift+Space`
   - 观察状态显示和点击计数

5. **停止点击**
   - 点击"停止点击"按钮
   - 或使用快捷键 `Ctrl+Shift+Space`
   - 或按 `ESC` 键（在选择模式下）

### 高级功能

#### 批量选择
1. 点击"批量选择按钮"
2. 在网页中依次点击多个要自动点击的元素
3. 已选择的元素会显示蓝色边框
4. 再次点击已选择的元素可以取消选择
5. 按 `ESC` 键完成批量选择

#### 保存和加载配置
- **保存配置**: 点击"保存配置"保存当前的设置和选择的元素
- **加载配置**: 点击"加载配置"恢复之前保存的配置

### 点击类型说明

- **普通点击**: 模拟鼠标左键点击
- **双击**: 模拟鼠标双击
- **按下**: 只触发鼠标按下事件
- **释放**: 只触发鼠标释放事件

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+Space` | 开始/停止自动点击 |
| `ESC` | 取消元素选择或停止点击 |

## 🎯 使用场景

- **游戏辅助**: 自动点击游戏中的按钮或物品
- **网页测试**: 重复点击表单提交按钮进行测试
- **数据录入**: 自动点击重复的操作按钮
- **网页抢票**: 快速重复点击购买按钮
- **签到打卡**: 自动点击签到按钮
- **广告点击**: 自动点击广告或链接

## ⚠️ 使用注意事项

1. **合法使用**: 请确保使用本扩展符合网站的服务条款
2. **适量使用**: 避免过于频繁的点击影响网站正常运行
3. **网站兼容性**: 某些网站可能有反自动化措施
4. **浏览器权限**: 扩展需要访问当前标签页的权限
5. **元素变化**: 如果网页结构改变，可能需要重新选择元素

## 🔧 故障排除

### 扩展无法加载
- 确保所有文件都在同一文件夹中
- 检查 `manifest.json` 文件格式是否正确
- 尝试重新加载扩展

### 无法选择元素
- 刷新网页后重试
- 检查网页是否完全加载
- 确保没有其他扩展干扰

### 点击不生效
- 确认选择的元素仍然存在
- 尝试不同的点击类型
- 检查网页是否有特殊的事件处理

### 扩展图标不显示
- 手动添加图标文件到扩展文件夹
- 重新加载扩展
