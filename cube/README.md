# 夹缝求生游戏

## 游戏说明

这是一个基于HTML5的"夹缝求生"游戏，玩家需要在上下蓝色区域合拢时躲在唯一的空缺位置。游戏采用了复杂的凹凸互补设计，上下区域有多个交错的凹凸结构，合拢时绝大部分区域完美契合，只留出一个明显的空缺区域供玩家躲避，每轮空缺位置随机变化。

## 游戏玩法

1. **目标**: 当上下蓝色区域合拢时，确保红色玩家角色完全位于唯一的空缺位置
2. **控制**: 
   - 使用屏幕上的左右箭头按钮
   - 或使用键盘箭头键 (←/→) 
   - 或使用 A/D 键
   - 移动设备支持滑动操作
3. **得分**: 每次成功躲避得1分
4. **难度**: 随着分数增加，合拢速度会逐渐加快
5. **安全区域**: 每轮随机出现的单一空缺位置（左侧、中心或右侧）

## 游戏特色

- 🧩 **复杂凹凸互锁设计**: 多个交错的凹凸结构，绝大部分区域完美契合
- 🎯 **单一空缺挑战**: 每轮只有一个安全区域，极具挑战性
- 🎲 **随机空缺位置**: 3种形状变体，空缺位置随机出现在左侧、中心或右侧
- 👁️ **视觉识别挑战**: 需要快速观察复杂形状来识别唯一的空缺位置
- 🎮 简单直观的游戏机制
- 📱 支持移动设备触摸操作
- 🏆 本地最高分记录保存
- 🎨 精美的渐变背景和立体阴影效果
- ⚡ 渐进式难度增加
- 🔥 纯粹的形状观察和精确定位

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 技术栈

- HTML5
- CSS3 (动画、渐变、响应式设计)
- Vanilla JavaScript (ES6+)
- LocalStorage (保存最高分)

## 游戏机制

- **复杂形状设计**: 上下区域各有多个凹凸结构，交错排列形成复杂的互锁形状
- **3种形状变体**: 
  - 变体1: 左侧空缺 (0%-15%)
  - 变体2: 中心空缺 (35%-55%) 
  - 变体3: 右侧空缺 (75%-100%)
- **随机生成**: 每轮随机选择一种形状变体，确保空缺位置不可预测
- **空缺识别**: 没有视觉指示器，需要仔细观察上下形状的不匹配区域
- **视觉线索**: 立体阴影和渐变效果帮助识别凹凸结构
- **初始速度**: 2秒合拢一次
- **最快速度**: 0.8秒合拢一次  
- **速度递增**: 每成功躲避一次，速度增加0.05秒
- **碰撞检测**: 玩家必须完全位于当前变体的唯一空缺区域内才算安全
- **观察挑战**: 完全依靠玩家对复杂形状的快速观察和精确判断
